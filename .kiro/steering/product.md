# Product Overview

3D熱拡散シミュレーション (Heat3D)。半導体デバイスの熱設計・解析を支援するための、高速かつ高精度な数値シミュレーション・ツールです。

## Core Capabilities

- **多次元熱拡散計算**: PBiCGStab, CG, SOR などの高度な反復法を用いた 3D 熱分布解析。
- **非一様グリッド対応**: 非一様グリッド (Non-uniform grid) を用いた、計算負荷の最適化と重要領域の解像度向上。
- **材料特性シミュレーション**: シリコン、銅、ハンダ、FR4 などの多様な材料の熱特性 (熱伝導率、比熱、密度) を考慮。
- **柔軟な設定管理**: JSON 形式の設定ファイルによる、ジオメトリ、材料、レイヤー構成の容易な変更。

## Target Use Cases

- **半導体パッケージ解析**: 積層構造を持つチップやパッケージ内の熱分布予測。
- **TSV 最適化**: Through-Silicon Via (TSV) の配置やサイズが熱特性に与える影響の評価。

## Value Proposition

複雑な 3D 構造、特に微細な TSV や積層チップを含む最新の半導体パッケージにおいて、現実的な時間で精度の高い熱解析結果を提供します。
